GenJAX Tutorial Visualization Images

This directory contains visualization images for the GenJAX tutorial chapters.

Image Generation Guide

All images should be generated by running the code snippets from the tutorial chapters. Below is a guide for each required visualization.

Chapter 2: Your First GenJAX Model

Location in tutorial: After line 265 (bar chart code)

Image to generate: first_model_outcome_distribution.png

  • Source code: Lines 244-265 in 02_first_model.md
  • What it shows: Bar chart of four outcomes (HH, HT, TH, TT) from 10,000 simulations
  • Expected appearance: Four bars of roughly equal height (~2500 each)
  • How to generate:

Image Specifications

All generated images should follow these specifications:

  • Format: PNG
  • DPI: 150 (good balance of quality and file size)
  • Size: Matplotlib default (typically 8x5 inches for single plots, 12x5 for side-by-side)
  • Style: Clean, professional appearance matching the tutorial’s friendly tone
  • Colors: Use the color schemes defined in the code (e.g., ‘#4ecdc4’ for green, ‘#6c5ce7’ for blue)
