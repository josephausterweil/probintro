## Quick Start: Generate All Images

To generate all visualization images:


## Prerequisites

The script requires:
- Python 3.8+
- JAX
- GenJAX (version 0.9.3 or 0.10.3+)
- matplotlib

Install dependencies:
```bash
pip install jax genjax matplotlib
```

## Image Specifications

All generated images follow these specifications:
- **Format:** PNG
- **DPI:** 150 (good balance of quality and file size)
- **Naming:** Descriptive lowercase with underscores
- **Colors:** Consistent with tutorial style

## Manual Generation (Alternative)

If you prefer to generate images manually:

1. Open the tutorial markdown files
2. Find the code blocks before each image placeholder
3. Run the code in a Jupyter notebook or Python script
4. Add `plt.savefig('content/genjax/images/<name>.png', dpi=150, bbox_inches='tight')` before `plt.show()`
5. Run to generate the image

## Troubleshooting

**Import errors:**
```bash
# Make sure GenJAX is installed correctly
pip install --upgrade genjax
```

**Image not appearing in Hugo:**
- Verify image path is `images/<filename>.png` (relative to the markdown file)
- Check that images are in `content/genjax/images/` directory
- Rebuild the Hugo site: `hugo server`

**Script fails:**
- Check that you're running from the project root directory
- Ensure all dependencies are installed
- Check GenJAX version compatibility
